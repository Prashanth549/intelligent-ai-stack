# Intelligent AI Stack

A modern content platform exploring how Artificial Intelligence is integrated into SaaS products, data systems, and real-world applications.

Live Website: https://www.intelligentaistack.com

---

## Overview

Intelligent AI Stack is a content-driven platform focused on breaking down how AI is transforming modern software systems.

The platform covers:

* AI in SaaS products
* AI-powered workflows
* Real-world AI applications
* System design and architecture

This project was built to combine **technical execution + content strategy**, creating a scalable foundation for a tech-focused publication.

---

## 🧠 Key Features

* 📰 Dynamic article system (Markdown-based)
* ⚡ Fast and modern frontend using React + Vite
* 🎨 Clean UI with Tailwind CSS
* 🔗 Multi-page routing with React Router
* 📈 Analytics integration (Vercel Analytics)
* 🌍 Deployed on Vercel with custom domain

---

## 🏗️ Tech Stack

**Frontend**

* React (TypeScript)
* Vite

**Styling**

* Tailwind CSS

**Routing**

* React Router

**Content System**

* Markdown + React Markdown

**Deployment**

* Vercel
